# Мониторинг

Комплексный обзор наблюдаемости SGroups: метрики Prometheus для сервера и агента,
распределенный трейсинг через OpenTelemetry, проверки здоровья и Grafana-дашборды.

## Метрики sg-server

### Эндпоинт

<CodeBlock>
  {dedent`
    GET http://<gateway-address>/metrics
  `}
</CodeBlock>

По умолчанию — `http://localhost:9100/metrics`.

## Конфигурация Prometheus

<CodeBlock language="yaml">
  {dedent`
    scrape_configs:
      - job_name: "sg-server"
        scrape_interval: 15s
        static_configs:
          - targets: ["sg-server:9100"]

      - job_name: "sg-agent"
        scrape_interval: 15s
        static_configs:
          - targets: ["agent-01:9650", "agent-02:9650"]
  `}
</CodeBlock>

## OpenTelemetry-трейсинг

sg-server поддерживает распределенный трейсинг через OpenTelemetry SDK.

<CodeBlock language="yaml">
  {dedent`
    tracing:
      enabled: true
      exporter: "otlp"
      endpoint: "otel-collector:4317"
  `}
</CodeBlock>

<table>
  <thead>
    <tr>
      <th>Экспортер</th>
      <th>Значение</th>
      <th>Протокол</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Jaeger</td>
      <td><code>jaeger</code></td>
      <td>HTTP Thrift (<code>/api/traces</code>)</td>
    </tr>
    <tr>
      <td>OTLP</td>
      <td><code>otlp</code></td>
      <td>gRPC (4317) или HTTP (4318)</td>
    </tr>
  </tbody>
</table>

Трейсятся: все gRPC-вызовы, SQL-запросы, межсервисное взаимодействие.

## Проверки здоровья (Health Check)

<CodeBlock language="bash">
  {dedent`
    # gRPC
    grpcurl -plaintext localhost:9000 hbf.v1.StatusService/Status

    # REST
    curl http://localhost:9100/v1/status
  `}
</CodeBlock>

### Интеграция с Docker / Kubernetes

<CodeBlock language="yaml">
  {dedent`
    # Docker Compose
    healthcheck:
      test: ["CMD", "curl", "-f", "http://localhost:9100/v1/status"]
      interval: 10s
      timeout: 5s
      retries: 3
  `}
</CodeBlock>

<CodeBlock language="yaml">
  {dedent`
    # Kubernetes
    livenessProbe:
      httpGet:
        path: /v1/status
        port: 9100
      initialDelaySeconds: 5
      periodSeconds: 10
  `}
</CodeBlock>

## Пример алертов

<CodeBlock language="yaml">
  {dedent`
    groups:
      - name: sgroups
        rules:
          - alert: SGServerHighErrorRate
            expr: rate(grpc_server_handled_total{grpc_code!="OK"}[5m]) > 0.1
            for: 5m
            labels:
              severity: warning
            annotations:
              summary: "Высокий уровень ошибок gRPC на sg-server"

          - alert: SGAgentSyncFailing
            expr: time() - sg_agent_sync_last_success_timestamp > 300
            for: 5m
            labels:
              severity: warning
            annotations:
              summary: "sg-agent не синхронизируется более 5 минут"

          - alert: SGAgentDisconnected
            expr: sg_agent_grpc_connected == 0
            for: 2m
            labels:
              severity: critical
            annotations:
              summary: "sg-agent потерял связь с sg-server"
  `}
</CodeBlock>
