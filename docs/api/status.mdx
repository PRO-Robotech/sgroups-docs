# Status API

API для получения информации о статусе синхронизации SGroups.

## Получение статуса

<span className="api-method api-method--get">GET</span> <code className="api-endpoint">/v1/status/list</code>

Возвращает текущий статус последней синхронизации.

### Пример запроса

<CodeBlock language="bash">
  {dedent`
    curl -X GET http://localhost:9006/v1/status/list \\
      -H 'Content-Type: application/json'
  `}
</CodeBlock>

### Ответ

<table>
  <thead>
    <tr>
      <th>Параметр</th>
      <th>Тип</th>
      <th>Описание</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>updatedAt</code></td>
      <td><code>{TYPES.timestamp}</code></td>
      <td>Время последнего успешно завершенного обновления БД.</td>
    </tr>
  </tbody>
</table>

### Пример ответа

<CodeBlock language="json">
  {dedent`
    {
      "updatedAt": "2025-01-15T10:30:00Z"
    }
  `}
</CodeBlock>

## Наблюдение за статусом (Watch)

<span className="api-method api-method--get">GET</span> <code className="api-endpoint">/v1/status/watch</code>

Серверный стриминг — возвращает поток обновлений статуса синхронизации в реальном времени.

### Пример запроса

<CodeBlock language="bash">
  {dedent`
    curl -X GET http://localhost:9006/v1/status/watch \\
      -H 'Content-Type: application/json'
  `}
</CodeBlock>

### Ответ

Поток JSON-объектов, каждый из которых содержит время последнего успешно завершенного обновления:

<CodeBlock language="json">
  {dedent`
    {
      "updatedAt": "2025-01-15T10:35:00Z"
    }
  `}
</CodeBlock>
