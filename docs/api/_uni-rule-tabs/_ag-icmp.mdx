Локальная сторона — адресная группа (`ADDRESS_GROUP`), транспорт — ICMP.
Удаленная сторона не указывается: правило применяется к ICMP-трафику выбранной адресной группы.

:::warning Transport
Блок `spec.transport` **обязателен**. Для типа правила AG-ICMP значение `spec.transport.protocol` должно быть `ICMP`, а записи задаются через `types`.
:::
