Локальная сторона — адресная группа, удаленная — доменное имя (`FQDN`).
Поле `remote.value` **обязательно** — содержит доменное имя. Поля `remote.name` и `remote.namespace` не используются.

:::warning Transport и Traffic
Блок `spec.transport` **обязателен**. Направление трафика — только `EGRESS` (AG отправляет трафик на внешний домен).
:::

          {
            "metadata": {
              "name": "allow-web-to-external-api",
              "namespace": "production"
