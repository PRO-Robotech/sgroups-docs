Локальная сторона — сервис, удаленная — IP-подсеть (`CIDR`).
Поле `remote.value` **обязательно**. Поля `remote.name` и `remote.namespace` не используются.

:::warning Transport и Traffic
Блок `spec.transport` **обязателен**. Направление трафика — `INGRESS` или `EGRESS` (значение `BOTH` недопустимо для CIDR).
:::
