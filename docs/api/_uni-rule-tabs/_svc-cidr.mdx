Локальная сторона — сервис (`SERVICE`), удаленная — IP-подсеть (`CIDR`).
Поле `remote.value` **обязательно**. Блок `spec.transport` задается явно и нужен
как для `EGRESS`, так и для `INGRESS`.


Service отправляет трафик к CIDR. Транспорт задается явно в правиле.
