## List

<ApiOperation method="POST" endpoint="/v1/address-groups/list">

Возвращает список групп адресов с фильтрацией. Ответ включает массив `refs[]` со связанными ресурсами.

#### Тело запроса

  <thead>

    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>selectors[].fieldSelector.name</code></td>
      <td>нет</td>
      <td><code>{TYPES.string}</code></td>
      <td>{DICTIONARY.name.short}</td>
    </tr>
    <tr>
      <td><code>selectors[].fieldSelector.namespace</code></td>
      <td>нет</td>
      <td><code>{TYPES.string}</code></td>
      <td>{DICTIONARY.namespace.short}</td>
    </tr>
    <tr>
      <td><code>selectors[].fieldSelector.refs[]</code></td>
      <td>нет</td>
      <td><code>{TYPES.resourceRef}</code></td>
      <td>{DICTIONARY.refs.short}</td>
    </tr>
    <tr>
      <td><code>selectors[].labelSelector</code></td>
      <td>нет</td>
      <td><code>{TYPES.mapStringString}</code></td>
      <td>{DICTIONARY.labelSelector.short}</td>
    </tr>
  </tbody>
</table>

#### Примеры запроса

:::info Правила комбинирования
- Внутри одного `selectors[]` элемента `fieldSelector` и `labelSelector` объединяются по **И** (AND).
- Между элементами массива `selectors[]` действует логическое **ИЛИ** (OR).
- Элементы массива `refs[]` внутри одного `fieldSelector` объединяются по **И** — AG должна быть связана со всеми перечисленными ресурсами.
:::
