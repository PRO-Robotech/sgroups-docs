:::info Правила комбинирования
- Внутри одного `selectors[]` элемента `fieldSelector` и `labelSelector` объединяются по **И** (AND).
- Между элементами массива `selectors[]` действует логическое **ИЛИ** (OR).
- Поля `addressGroup` и `network` в `fieldSelector` — это объекты `{name, namespace}`.
:::
