# UniRule

{DICTIONARY.resourceUniRule.full}

:::info Общие поля всех комбинаций
Независимо от типа эндпоинтов каждое правило содержит:
- `metadata.name`, `metadata.namespace` — обязательные идентификаторы
- `spec.action` — `ALLOW` или `DENY` (обязательно)
- `spec.session.traffic` — `BOTH`, `INGRESS` или `EGRESS`
:::

:::warning Блок transport зависит от комбинации эндпоинтов и направления трафика
Если одна из сторон правила — ресурс Service, то `transport` может наследоваться из его `transports`:

| Комбинация | Traffic | transport в правиле |
|---|---|---|
| AG → AG | любой | **обязателен** |
| AG → SVC | egress / both | не нужен (из remote Service) |
| AG → SVC | ingress | **обязателен** |
| SVC → AG | egress / both | **обязателен** |
| SVC → AG | ingress | не нужен (из local Service) |
| SVC → SVC | любой | не нужен (из обоих Service) |
| AG → FQDN | любой | **обязателен** |
| SVC → FQDN | любой | **обязателен** |
| AG → CIDR | любой | **обязателен** |
| SVC → CIDR | любой | **обязателен** |
:::

## Delete

<ApiOperation method="POST" endpoint="/v1/rules/delete">

Удаляет правило по `uid` или по комбинации `name` + `namespace`. Операция общая для всех типов правил.

#### Тело запроса

<table>
  <thead>
    <tr>
      <th>Параметр</th>
      <th>Обязательность</th>
      <th>Тип</th>
      <th>Описание</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>uid</code></td>
      <td>нет*</td>
      <td><code>string</code></td>
      <td>{DICTIONARY.uid.short}</td>
    </tr>
    <tr>
      <td><code>name</code></td>
      <td>нет*</td>
      <td><code>string</code></td>
      <td>{DICTIONARY.name.short}</td>
    </tr>
    <tr>
      <td><code>namespace</code></td>
      <td>нет*</td>
      <td><code>string</code></td>
      <td>{DICTIONARY.namespace.short}</td>
    </tr>
  </tbody>
</table>

> *Необходимо указать `uid` **или** комбинацию `name` + `namespace`.

#### Удаление по uid

<CodeBlock language="bash">
  {dedent`
    curl -X POST http://localhost:9006/v1/rules/delete \\
      -H 'Content-Type: application/json' \\
      -d '{
        "rules": [
          {
            "metadata": {
              "uid": "c9d0e1f2-a3b4-5678-cdef-789012345678"
            }
          }
        ]
      }'
  `}
</CodeBlock>

#### Удаление по name + namespace

<CodeBlock language="bash">
  {dedent`
    curl -X POST http://localhost:9006/v1/rules/delete \\
      -H 'Content-Type: application/json' \\
      -d '{
        "rules": [
          {
            "metadata": {
              "name": "allow-web-to-db",
              "namespace": "production"
            }
          }
        ]
      }'
  `}
</CodeBlock>

#### Пример ответа

<CodeBlock language="json">
  {dedent`
    {}
  `}
</CodeBlock>
